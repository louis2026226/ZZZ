【只做一次】GitHub 仓库 → Settings → Secrets and variables → Actions → New repository secret

DEPLOY_HOST     8.134.168.87
DEPLOY_USER     root
DEPLOY_PATH     /root/louis-zz
DEPLOY_SSH_KEY  （整段私钥，含 BEGIN/END 行）

服务器上生成专用于 Actions 的密钥（勿用个人日常私钥）：
ssh-keygen -t ed25519 -f /root/.ssh/github_actions -N ""
cat /root/.ssh/github_actions.pub >> /root/.ssh/authorized_keys
cat /root/.ssh/github_actions

安全组放行 SSH 22（供 GitHub Runner 连入）。

【可选 ECS】在 GitHub → Settings → Variables → 新建变量 DEPLOY_SSH_ENABLED = true，并配齐下面 4 个 Secret 后，每次 push main 会自动 SSH 部署。
不配变量则本 workflow 跳过（不红叉）。只用 Railway 不必配。
