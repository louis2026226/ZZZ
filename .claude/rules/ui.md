# UI / 角色权限模块记忆

## 角色区分
- `isHost`：`bUsername === hostUsername`，房主身份
- B端（Admin）页面：`/b/dashboard` → `AdminRoom.jsx`
- C端（Player）页面：`/c/play` → `PlayerRoom.jsx`（或同文件内分支渲染）

## 房主（isHost）专属 UI
- 显示：**开始答题** / **结束答题** 按钮（原名：上课/下课，v1.0.99 改名）
- 显示：开奖弹窗（选择 1-4 结算）
- 显示：🔔 铃铛按钮（下注阶段提醒玩家）
- **隐藏**：选号区（1-4 号码按钮）
- **隐藏**：选积分区（金额按钮 + 自选输入框）
- **隐藏**：确定按钮

## 非房主（普通玩家）UI
- 显示：选号区、选积分区、确定按钮
- 隐藏：开始答题/结束答题按钮
