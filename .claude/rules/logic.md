## 新增功能

### 红包功能
- **B端房主**：在消息板底部右侧添加红包图标(hb2.png)
- **C端玩家**：在消息板底部右侧添加红包图标(hb2.png)
- **点击效果**：点击红包���标后，在消息板内显示红包图片(hb1.png)
- **自动消失**：红包图片5秒后自动隐藏

### 技术实现
1. **MessageBoard组件**：
   - 新增 `showRedPacket` 状态控制红包显示
   - 在消息板容器内绝对定位红包图片
   - 支持自定义图片显示逻辑

2. **AdminRoom组件**：
   - 添加 `showRedPacket` 状态管理
   - 添加红包按钮和点击事件处理
   - 添加5秒自动隐藏逻辑
   - 房主专属按钮（固定定位在右下角）

3. **PlayerRoom组件**：
   - 添加 `showRedPacket` 状态管理
   - 添加红包按钮和点击事件处理
   - 添加5秒自动隐藏逻辑
   - 所有玩家可用（固定定位在右下角）

4. **版本更新**：
   - 版本号同步更新至 1.0.108
   - BLogin.jsx 和 CLogin.jsx 版本信息更新

### 注意事项
- 红包图片来自 public 文件夹路径 (`/hb1.png`, `/hb2.png`)
- 使用绝对定位确保红包图片始终显示在消息板右下角
- 红包图片大小限制为最大180px宽度和高度
- 避免干扰现有消息显示
